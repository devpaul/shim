raw: function = (<any> String).raw
  • Defined in native/string.ts:39

A tag function for template strings to get the template string's raw string form.

param

Call site object (or a template string in TypeScript, which will transpile to one)

param

Values to substitute within the template string (TypeScript will generate these automatically)

returns

String containing the raw template string with variables substituted

example

// Within TypeScript; logs 'The answer is:\n42' let answer = 42; console.log(string.rawThe answer is:\n${answer});

// The same example as above, but directly specifying a JavaScript object and substitution console.log(string.raw({ raw: [ 'The answer is:\n', '' ] }, 42));
