isSafeInteger

  • isSafeInteger(value: any): boolean
    • Defined in number.ts:58

    Determines whether the passed value is an integer that is 'safe,' meaning:

    1. it can be expressed as an IEEE-754 double precision number
    2. it has a one-to-one mapping to a mathematical integer, meaning its IEEE-754 representation cannot be the result of rounding any other integer to fit the IEEE-754 representation

    Parameters

    • value: any

      The value to test

    Returns boolean

    true if the value is an integer, false if it is not
